## TicTacToe
A powerful, single-file Todo List.

## Features
- Add tasks
- Edit tasks
- Remove Tasks
- Progress Bar
- Theme Mode Toggle

## Demo
![TodoList Demo](assets/demo.PNG)

## How to run
1. `dotnet restore`
2. `dotnet build`
3. `dotnet run`

## Technologies
- HTML
- CSS
- JavaScript
- C#
- JSON
